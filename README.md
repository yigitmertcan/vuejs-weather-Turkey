# vuejs-weather-Turkey
 Vuejs Weather App forTurkey

## Project setup
```
npm install
```
### Compiles and hot-reloads for development
```
npm run serve
```
### Compiles and minifies for production
```
npm run build
```
### Lints and fixes files
```
npm run lint
```
<br>
Axios library for api connection.<br>
Css Framework is Boostrap.<br>
<br>
Api: collectapi.com<br>
<br>
Demo : [a link](http://mertcanyigit.site/weather)
